Less Info

The general idea is that on the YouBeHero site, all shop affiliate links will have these 4 queryString parameters added to them (server-side) on a per-user basis:

  • ubhTr
  • commission_percentage
  • cause-name
  • cause-logo-url

So, if someone comes to this landing page from the shop links on the YouBeHero website, the url should contain (valid) values for these 4 parameters.

If this info is present, it will be stored in localStorage for later use, along with:

  • a unique transaction ID
  • the eshop name (or url)
  • a timestamp

Try it out with some sample values! (Notice: After clicking, you will be redirected back here with sample parameter values in the queryString of the URL. The "ubhTr" parameter is a Unique-User-ID (UUID) issued by the server and may (or may not) expire after a certain period of time, or if it is used to register a transaction)
